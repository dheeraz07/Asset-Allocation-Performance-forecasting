# Raw data

The three large CSVs used by this project are available on the challenge
page at https://challengedata.ens.fr/participants/challenges/167/
under the "Dataset files" section.

Download and place them at the project root (one level above this
directory), keeping the original filenames:

- `X_train_9xQjqvZ.csv`
- `X_test_1zTtEnD.csv`
- `y_train_Ppwhaz8.csv`

The files are intentionally excluded from git via `.gitignore` because
`X_train_9xQjqvZ.csv` is 272 MB, well above GitHub's 100 MB hard limit
for versioned files.
